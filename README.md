# Subject-Manager
Gestor de Disciplina em linguagem C

Aplicação desenvolvida em C, que permite a gestão de uma disciplina. Ajuda tanto alunos, como professores. Usou-se listas ligadas, apontadores e alocação de memória (malloc).
Abaixo, são apresentadas as principais funcionalidades e respetivos menus de Aluno e Professor. Por defeito já se encontram alguns alunos e professores inseridos.

Exemplo Login: 1111111, "123456" -> Professor
	   	    8090228, "123456" -> Aluno Luis
